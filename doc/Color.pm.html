NAME

Color.pm - Farben und Konvertierungsfunktionen

SYNOPSIS

    use VRML::Color;
    my $color = rgb_color('red');
      ODER mit dem gleichen Ergebnis
    my $color = rgb_color('FF0000');
      ODER mit dem gleichen Ergebnis
    my $color = rgb_color('255 0 0');
      Natürlich geht auch
    my $color = rgb_color('1 0 0');

Es können auch prozentuale Anteile einer Farbe angeben werden (Trennzeichen _ oder %).

        red_40 = '0.4 0 0'
        yellow%30 = '0.3 0.3 0'
        gray%30 = '0.7 0.7 0.7' !!!

BUGS

Grün wird in X11 durch den Wert 0x008000 dargestellt. Für VRML ist jedoch 0x00FF00 besser geeignet. Deshalb wird in diesem Modul für 'green' der Wert '0 1 0' generiert anstatt '0 0.5 0'.
