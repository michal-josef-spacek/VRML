VRML.pm

NAME

VRML - implements VRML Nodes independent of specification (1.x or 2.0)

SYNOPSIS

use VRML; $vrml = new VRML; $vrml->browser('Netscape+Live3D'); $vrml->at('5 3 1'); $vrml->cube('-50 -10 0','yellow'); $vrml->end; $vrml->print; OR with the same result use VRML; VRML ->browser('Netscape+Live3D') ->at('5 3 1')->cube('-50 -10 0','yellow')->end ->print;

DESCRIPTION

  • $content_type; # for CGI scripts only

    content_type is 'x-world/x-vrml' for VRML 1.0

    or 'model/vrml' for VRML 2.0

.
