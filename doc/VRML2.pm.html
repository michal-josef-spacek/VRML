NAME

VRML::VRML2.pm - VRML Methoden für den VRML 2.0/VRML97 Standard

SYNOPSIS 

  use VRML::VRML2;


  $vrml = new VRML::VRML2;
  $vrml->browser('Cosmo Player 2.0','Netscape');
  $vrml->at('-15 0 20');
  $vrml->box('5 3 1','yellow');
  $vrml->back;
  $vrml->print;
  $vrml->save;


  ODER mit dem gleichen Ergebnis


  use VRML::VRML2;


  VRML::VRML2->new
  ->browser('Cosmo Player 2.0','Netscape')
  ->at('-15 0 20')->box('5 3 1','yellow')->back
  ->print->save;

DESCRIPTION

Die Beschreibung der implementierten VRML-Methoden befindet sich in dem Modul VRML, die der Basis-Methoden im Modul VRML::Base.
